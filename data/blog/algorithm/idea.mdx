## 截图软件

写博文时有一个痛点是：**当有个截图，需要上传到 github 图床时，第一步截图，第二步到 picgo 里点击上传剪切板。**

1. 如果有这么个截图软件能截图后能一键选择上传到图床岂不是很方便？
2. 鉴于现在 AI 制图如此火爆，能否接入 api 对截图的部分进行 AI 转换（当然可以加一点 prompt 辅助），这样可以直接出图，并且也能有效解决版权问题。

看了会儿 `tauri` 的文档，感觉有可行性，但是 rust 不太熟，后续学习了 rust 再来看看这个想法怎么落地吧。

PS: 用过的 ishot pro、 shottr 都是收费软件，咱要做就做一个纯开源的~
